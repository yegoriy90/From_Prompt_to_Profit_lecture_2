Prompt Engineering 2.0

Шпаргалка для лекції по генерації контенту в ШІ

1
Завдання (Job)
обов'язково ▼
Перший рядок — одне речення: що саме потрібно, для кого, де буде використано. Це якір для всього промпту.
Слабко:
"woman"
Сильно:
"Hero image для Instagram-реклами кавового бренду: жінка з кавою вранці вдома"
2
Суб'єкт + деталі
обов'язково ▼
4–6 ключових деталей: матеріал, фактура, поза, вік, емоція, одяг, дія. Кожна деталь — коротка фраза, не одне слово.
Слабко:
"red fox"
Сильно:
"руда лисиця, злегка вологе хутро, видихає пару в холодному повітрі, насторожений погляд вліво"
3
Стиль + медіум
важливо ▼
Художній напрямок, техніка або камера. Можна поєднувати стилі: "ренесансний живопис + сучасна фотографія".
Слабко:
"realistic"
Сильно:
"medium format camera, shallow depth of field, editorial photography style, desaturated tones"
4
Освітлення
важливо ▼
Освітлення — половина настрою зображення. Вкажи джерело, напрямок, якість та колірну температуру.
Слабко:
"good lighting"
Сильно:
"м'яке бічне освітлення від вікна зліва, золота година, довгі тіні, warm 4000K tones"
5
Композиція + кадр
важливо ▼
Кут знімання, відстань, правило третин, негативний простір. Особливо важливо для Product Shot і портрету.
Слабко:
"close up"
Сильно:
"low angle shot, суб'єкт у лівій третині кадру, розмитий нейтральний фон, 85mm portrait lens"
6
Технічні параметри
опційно ▼
Роздільність, aspect ratio, негативний промпт (для SD), вагові коефіцієнти. Синтаксис залежить від моделі.
Слабко:
"8k ultra hd"
Сильно:
"--ar 4:5 --style raw --no overexposed, blurry, watermark (Midjourney)"
📐
Базовий
Структурований промпт
Чітка послідовність: Завдання → Суб'єкт → Стиль → Освітлення → Композиція → Параметри.
"Hero shot кросівок Nike: білий кросівок на мокрому асфальті вночі, відбиття неону, overhead shot, контровий LED, --ar 1:1"
🔗
Базовий
Reference-based промпт
Прикріплюєш зображення як reference + описуєш що саме перенести: стиль, освітлення, позу, але не людину.
"[ref image] — використай тільки стиль освітлення та колірну гаму, намалюй нового персонажа"
🎭
Середній
Style stacking
Поєднання двох несумісних стилів для унікального результату. Ренесанс + cyberpunk, акварель + 3D render.
"Портрет у стилі голландського живопису XVII ст. з елементами cyberpunk: масляна текстура + неонові відблиски"
🧪
Середній
Negative prompting
SD: explicit negative prompt. GPT/MJ: описуй чого НЕ хочеш через позитив — "чисте небо" замість "--no clouds".
"Natural skin tones, matte finish — без дефектів, без штучних кольорів, без перенасиченості"
🔢
Середній
Вагові коефіцієнти
Midjourney: слово::2 підсилює. SD: (слово:1.5). Балансуєш між елементами сцени.
"sunset::2 beach::1 woman::0.8 — захід сонця домінує, жінка другорядна"
🔄
Середній
Iterative prompting
Діалог з моделлю: генеруєш базу → описуєш що змінити → уточнюєш. Найефективніше у GPT Image та Nano Banana.
"[після результату]: залиш ту саму композицію, але зміни освітлення на вечірнє та прибери фон"
🧩
Просунутий
Prompt chaining
Генерація кількох зображень як єдиної системи: спочатку фон, потім персонаж, потім Inpainting.
"Крок 1: тільки фон. Крок 2: персонаж на білому. Крок 3: compositing через Inpainting"
🤖
Просунутий
AI-assisted prompting
Просиш ChatGPT або Claude написати промпт для іншої моделі. Особливо ефективно для Midjourney і FLUX.
"Напиши Midjourney-промпт для luxury-парфумів: мінімалізм, осіння палітра, editorial style"
